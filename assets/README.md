# Ассеты для лендинга

Положите сюда файлы и при необходимости раскомментируйте разметку в `index.html`.

| Файл | Рекомендация |
|------|----------------|
| `appicon.png` | Иконка приложения, ~1024×1024 или квадрат, в `index.html` раскомментируйте `<img class="appicon" …>` и удалите/скройте `.appicon-fallback` при желании |
| `screen-0.png` | Главный скрин в hero (портрет, ~1179×2556 или масштабируемый PNG) |
| `screen-1.png` … `screen-3.png` | Дополнительные скриншоты для секции «Screens» — замените блоки `.screen-slot` на `<img …>` по аналогии с комментариями в HTML |

Имена можно изменить — тогда обновите пути в `index.html`.
